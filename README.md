# Tunic Text

Small web application that contains a keyboard for writing the text in tunic. I created this to help me translate the text throughout the game.

# Installation

This is basically vanilla HTML and CSS with JS that I opted to write in (and learn) Typescript. To get it running locally, you need to compile the typescript code first. Follow the [official docs](https://www.typescriptlang.org/docs/handbook/typescript-tooling-in-5-minutes.html) on how to set this up. Then you can simply open the index.htm file in a browser of your choice.
